# Proyek-Terapan-BPA
Project Management BPA (Planner U)

Project Management BPA (Planner U) merupakan aplikasi berbasis web yang dikembangkan untuk membantu pengelolaan tugas dan proyek di lingkungan BPA Telkom University. Sistem ini dirancang untuk mengatasi permasalahan pengelolaan tugas yang sebelumnya masih tersebar di berbagai platform seperti Microsoft Excel, Microsoft Planner, dan WhatsApp, sehingga kurang efisien dan sulit dipantau secara terintegrasi.

Aplikasi ini memungkinkan manajer dan staff untuk mengelola tugas, memantau progres pekerjaan, serta berkolaborasi dalam satu sistem terpusat. Dengan adanya fitur seperti dashboard, manajemen tugas, sub-task, penentuan PIC, serta notifikasi dan reminder deadline, sistem ini diharapkan dapat meningkatkan efisiensi kerja, transparansi, dan koordinasi tim.
a. fitur utama
- Dashboard monitoring tugas
- Manajemen tugas & sub-task
- Penentuan PIC (penanggung jawab)
- Deadline & reminder
- Notifikasi update progress
- Monitoring progress project
- Filter tugas (pribadi & tim)
b. role pengguna
- Manajer: monitoring, review, dan pengelolaan seluruh tugas
- Staff: mengerjakan tugas dan update progress
c. teknologi yang digunakan
- Laravel (Backend)
- MySQL / SQLite (Database)
- Web-based application
